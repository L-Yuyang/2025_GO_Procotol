Spatial transcriptomics (ST) offers unprecedented insights into the molecular architecture of tissues, yet robust methods for validating functional pathway activity remain critical. This study presents a computational framework for quantifying and validating pathway activity in ST data. By integrating Gene Set Enrichment Analysis (GSEA) for group-level validation and AUCell for single-spot resolution scoring, we provide a dual-layered approach to pathway analysis. 
Using the apoptotic pathway in Alzheimer's Disease (AD) as a representative example, we first load and preprocess the spatial omics data, standardize it, and identify highly variable genes to ensure comparability between groups. GSEA is applied to validate gene ontology (GO) pathways, and AUCell analysis is used to calculate enrichment scores (AUC) for each cell, providing insights into pathway activity at the single-cell level. The AUC scores are integrated into the Seurat object for further analysis. The apoptotic pathway, selected as an example, undergoes GSEA validation to illustrate the process. Statistical comparisons between the AD and control (Con) groups reveal higher apoptotic scores in the AD group. 
This methodology emphasizes the importance of GSEA validation for all pathways to ensure reliable and reproducible results, underscoring the robustness of pathway activity analyses in spatial omics studies.
